Chat that does things in your app — not just answers questions. Drop in a chat component, register a few tools, and your users get an AI assistant that can read your app's state, render rich UI inline, ask for approval before destructive actions, and call any backend you've built.

  <Step title="Wrap your app">
    ```tsx
    import { CopilotKit } from "@copilotkit/react-core/v2";

    <CopilotKit runtimeUrl="/api/copilotkit" agent="default">
      <YourApp />
    </CopilotKit>
    ```
  </Step>
  <Step title="Drop in the chat">
    ```tsx
    import { CopilotSidebar } from "@copilotkit/react-core/v2";

    <CopilotSidebar />
    ```
  </Step>
</Steps>

## Use any agent backend

CopilotKit defaults to the **Built-in Agent** — direct LLM access, zero framework. If you already have an agent in LangGraph, Mastra, or somewhere else, point CopilotKit at it instead.

## How it fits together

<Columns cols={3}>
  <Card title="React UI" icon="browser">
    `<CopilotKit>` provider, drop-in chat components, hooks (`useAgent`, `useFrontendTool`, `useComponent`). Same API across every backend.
  </Card>
  <Card title="Copilot Runtime" icon="server">
    A server endpoint (Hono / Express / Next.js) that bridges agents to the frontend over AG-UI events. Owns auth, middleware, threads, routing.
  </Card>
  <Card title="Your agent" icon="cpu">
    Runs wherever you want — built-in, LangGraph, Mastra, ADK, custom. Defines the brain.
  </Card>
</Columns>
