## Signature

```ts
function createCopilotExpressHandler(params: CopilotExpressEndpointParams): Router
```

Returns an Express `Router` pre-configured with all CopilotKit routes. Pass it to `app.use()` to mount.

## Parameters

| Name | Type | Required | Default | Description |
|---|---|---|---|---|
| `runtime` | `CopilotRuntimeLike` | Yes | — | A `CopilotRuntime`, `CopilotSseRuntime`, or `CopilotIntelligenceRuntime` instance. |
| `basePath` | `string` | Yes | — | Base path for all routes (e.g. `"/api/copilotkit"`). Must be non-empty. |
| `mode` | `"multi-route" \| "single-route"` | No | `"multi-route"` | Route layout. `"multi-route"` exposes separate REST routes; `"single-route"` uses a single POST with JSON envelope dispatch. |
| `cors` | `boolean \| CorsOptions` | No | `true` | CORS configuration. `true` = permissive (all origins, all methods, all headers). `false` = no CORS middleware applied. Object = passed directly to the `cors` npm package. |
| `hooks` | `CopilotRuntimeHooks` | No | — | Lifecycle hooks: `onRequest`, `onBeforeHandler`, `onResponse`, `onError`. |

## Routes exposed in `multi-route` mode

| Method | Path | Description |
|---|---|---|
| `GET` | `{basePath}/info` | Runtime info |
| `POST` | `{basePath}/agent/:agentId/run` | Start an agent run |
| `POST` | `{basePath}/agent/:agentId/connect` | Connect to a running agent |
| `POST` | `{basePath}/agent/:agentId/stop/:threadId` | Stop an agent run |
| `POST` | `{basePath}/transcribe` | Transcribe audio |

## Hooks (`hooks`)

| Hook | Signature | Description |
|---|---|---|
| `onRequest` | `(ctx: HookContext) => MaybePromise<Request \| void>` | Runs before routing. Return a modified `Request` or throw a `Response` to short-circuit. |
| `onBeforeHandler` | `(ctx: HandlerHookContext) => MaybePromise<Request \| void>` | Runs after routing. Receives resolved `route` info. |
| `onResponse` | `(ctx: ResponseHookContext) => MaybePromise<Response \| void>` | Runs after the handler. Return a modified `Response` or void. |
| `onError` | `(ctx: ErrorHookContext) => MaybePromise<Response \| void>` | Runs on error. Return a custom error `Response` or void for default behavior. |

## Example — basic Express setup

```ts
import express from "express";
import { CopilotRuntime, BuiltInAgent } from "@copilotkit/runtime/v2";
import { createCopilotExpressHandler } from "@copilotkit/runtime/v2";

const runtime = new CopilotRuntime({
  agents: {
    default: new BuiltInAgent({ model: "openai/gpt-5-mini" }),
  },
});

const app = express();

app.use(
  createCopilotExpressHandler({
    runtime,
    basePath: "/api/copilotkit",
  }),
);

app.listen(4000, () => console.log("Server running on port 4000"));
```

## Example — custom CORS

```ts
import { createCopilotExpressHandler } from "@copilotkit/runtime/v2";

app.use(
  createCopilotExpressHandler({
    runtime,
    basePath: "/api/copilotkit",
    cors: {
      origin: ["https://app.example.com", "https://staging.example.com"],
      credentials: true,
    },
  }),
);
```

## Example — auth hook with single-route mode

```ts
import { createCopilotExpressHandler } from "@copilotkit/runtime/v2";

app.use(
  createCopilotExpressHandler({
    runtime,
    basePath: "/api/copilotkit",
    mode: "single-route",
    hooks: {
      onRequest: async ({ request }) => {
        const token = request.headers.get("authorization");
        if (!token || !isValidToken(token)) {
          throw new Response("Unauthorized", { status: 401 });
        }
      },
    },
  }),
);
```
