# Prebuilt Sidebar

## What This Demo Shows

Docked `<CopilotSidebar />` chat alongside the main page content.

## How to Interact

The sidebar is open by default. Use the launcher button to toggle it closed/open.

## Technical Details

- `<CopilotSidebar />` is a pre-built chat component that docks to the side of the page.
- `defaultOpen={true}` keeps the sidebar visible on initial load so the form factor is obvious.
- This demo reuses the shared MS Agent Python backend (no agent-specific logic on the frontend).
