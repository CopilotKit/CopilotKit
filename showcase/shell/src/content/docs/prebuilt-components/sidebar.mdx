CopilotKit ships three prebuilt chat components that connect directly to your LangChain agent. Each is a wrapper around `CopilotChat` with a different layout — pick the one that fits your app and you're done.

<Callout type="info">
  If you've completed the [quickstart](/quickstart), you already have one of these set up.
</Callout>

<InlineDemo integration="langgraph-python" demo="prebuilt-sidebar" />

## Setup

Import the default styles in your root layout:

```tsx title="layout.tsx"
import "@copilotkit/react-ui/v2/styles.css";
```

### CopilotSidebar

Wraps your main content and provides a collapsible sidebar chat.

```tsx title="page.tsx"
// [!code word:CopilotSidebar]
import { CopilotSidebar } from "@copilotkit/react-core/v2";

export function YourApp() {
  return (
    <CopilotSidebar
      defaultOpen={true}
      labels={{
        modalHeaderTitle: "Sidebar Assistant",
        welcomeMessageText: "How can I help you today?",
      }}
    >
      <YourMainContent />
    </CopilotSidebar>
  );
}
```

## Customization

All three components support the [slot system](/custom-look-and-feel/slots) for deep customization — from Tailwind classes to full component replacement:

```tsx title="page.tsx"
<CopilotChat
  // Style slots with Tailwind classes
  input={{
    textArea: "text-lg",
    sendButton: "bg-blue-600 hover:bg-blue-700",
  }}
  // Customize nested message slots
  messageView={{
    assistantMessage: {
      className: "bg-gray-50 rounded-xl p-4",
      toolbar: "border-t mt-2",
    },
    userMessage: "bg-blue-100 rounded-xl",
  }}
  // Hide elements by returning null
  scrollView={{
    feather: () => null,
  }}
/>
```
