We're adding dedicated guides for every frontend. This frontend supports the same CopilotKit capabilities you see in the React docs: chat surfaces, frontend tools, generative UI, shared app context, human-in-the-loop flows, and AG-UI-compatible agents.

For now, use these docs in three layers:

1. Start with the Quickstart in this sidebar. It shows the install, provider, and first working connection for this frontend.
2. Use the React docs below as feature blueprints. The code samples are React-shaped, but the product patterns are the same ones to build here.
3. Keep the [Reference docs](/reference) open for exact APIs, package names, and configuration details.

When you want a feature from the React docs, ask your coding agent to translate that pattern into this frontend and point it at the reference docs for the exact API surface.
