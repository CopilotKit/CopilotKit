One of the most common use cases for CopilotKit is to register app state and context using `useAgentContext`.
This way, you can notify CopilotKit of what is going in your app in real time.
Some examples might be: the current user, the current page, etc.

This context can then be shared with your AG2 backend.

## Implementation
<Callout>
    Check out the [Frontend Data documentation](/langgraph/agent-app-context) to understand what this is and how to use it.
</Callout>

<Callout type="info">
  CopilotKit consumes AG-UI protocol events streamed by AG2 over <code>/chat</code>. See the <a href="https://docs.ag2.ai/latest/docs/user-guide/ag-ui/" target="_blank">AG2 AG-UI integration docs</a>.
</Callout>

            <Step>
                ### Add the data to the Copilot

                The [`useAgentContext` hook](/reference/v2/hooks/useAgentContext) is used to add data as context to the Copilot.

                ```tsx title="YourComponent.tsx" showLineNumbers {1, 7-10}
                "use client" // only necessary if you are using Next.js with the App Router. // [!code highlight]
                import { useAgentContext } from "@copilotkit/react-core/v2"; // [!code highlight]
                import { useState } from 'react';

                export function YourComponent() {
                  // Create colleagues state with some sample data
                  const [colleagues, setColleagues] = useState([
                    { id: 1, name: "John Doe", role: "Developer" },
                    { id: 2, name: "Jane Smith", role: "Designer" },
                    { id: 3, name: "Bob Wilson", role: "Product Manager" }
                  ]);

                  // Define Copilot readable state
                  // [!code highlight:4]
                  useAgentContext({
                    description: "The current user's colleagues",
                    value: colleagues,
                  });
                  return (
                    // Your custom UI component
                    <>...</>
                  );
                }
                ```
            </Step>

            <Step>
                ### Read the context from the request
                In AG2, readables arrive on `RunAgentInput.context` as a list of `{description, value}` items. Read them in your endpoint before dispatching:

                <Tabs groupId="language_ag2_readables_state" items={['Python']} default="Python" persist>
                    <Tab value="Python">
                        ```python title="agent.py"
                        from ag2.ag_ui import RunAgentInput

                        def get_readable(message: RunAgentInput, description: str) -> str | None:
                            return next(
                                (item.value for item in message.context if item.description == description),
                                None,
                            )
                        ```
                    </Tab>
                </Tabs>

            </Step>

            <Step>
                ### Consume the data in your AG2 backend
                Pass the readable into the run as a conversation variable, then expose it to the LLM through a tool. The `Variable()` annotation injects the value into the tool without the LLM having to pass it around.

                <Tabs groupId="language_ag2_readables_backend" items={['Python']} default="Python" persist>
                    <Tab value="Python">
                        ```python title="agent.py"
                        import json
                        from typing import Annotated

                        from fastapi import FastAPI, Header
                        from fastapi.responses import StreamingResponse

                        from ag2 import Agent, Variable, tool
                        from ag2.ag_ui import AGUIStream, RunAgentInput
                        from ag2.config import OpenAIConfig

                        def get_readable(message: RunAgentInput, description: str) -> str | None:
                            return next(
                                (item.value for item in message.context if item.description == description),
                                None,
                            )

                        @tool
                        def get_colleagues(
                            colleagues: Annotated[list, Variable(default_factory=list)],
                        ) -> list:
                            """Return the current user's colleagues from CopilotKit readables."""
                            return colleagues

                        agent = Agent(
                            name="assistant",
                            prompt=(
                                "You are a helpful assistant that can help emailing colleagues. "
                                "Call `get_colleagues` before suggesting recipients."
                            ),
                            config=OpenAIConfig(model="gpt-4o-mini"),
                            tools=[get_colleagues],
                        )

                        stream = AGUIStream(agent)
                        app = FastAPI()

                        @app.post("/chat")
                        async def run_agent(
                            message: RunAgentInput,
                            accept: str | None = Header(None),
                        ) -> StreamingResponse:
                            raw = get_readable(message, "The current user's colleagues")
                            colleagues = json.loads(raw) if raw else []
                            return StreamingResponse(
                                stream.dispatch(message, variables={"colleagues": colleagues}, accept=accept),
                                media_type=accept or "text/event-stream",
                            )
                        ```
                    </Tab>
                </Tabs>
            </Step>

            <Step>
                ### Give it a try
                Ask your agent a question about the context. It should be able to answer.
            </Step>

            <Step>
                ### Consume the data in your AG2 backend
                For a minimal backend, inject all readables directly into the agent's prompt for the run — no tools required:

                <Tabs groupId="language_ag2_readables_minimal" items={['Python']} default="Python" persist>
                    <Tab value="Python">
                        ```python title="agent.py"
                        from fastapi import FastAPI, Header
                        from fastapi.responses import StreamingResponse

                        from ag2 import Agent
                        from ag2.ag_ui import AGUIStream, RunAgentInput
                        from ag2.config import OpenAIConfig

                        agent = Agent(
                            name="assistant",
                            prompt="You are a helpful assistant.",
                            config=OpenAIConfig(model="gpt-4o-mini"),
                        )

                        stream = AGUIStream(agent)
                        app = FastAPI()

                        @app.post("/chat")
                        async def run_agent(
                            message: RunAgentInput,
                            accept: str | None = Header(None),
                        ) -> StreamingResponse:
                            # Surface every CopilotKit readable to the LLM as extra prompt context.
                            readables = [
                                f"{item.description}: {item.value}" for item in message.context
                            ]
                            return StreamingResponse(
                                stream.dispatch(message, prompt=readables, accept=accept),
                                media_type=accept or "text/event-stream",
                            )
                        ```
