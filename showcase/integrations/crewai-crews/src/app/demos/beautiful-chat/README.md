# Beautiful Chat

## What This Demo Shows

The flagship CopilotKit surface: a polished chat that combines nearly every generative-UI pattern against a single LangGraph (Python) agent.

- **Controlled Generative UI**: frontend-defined `pieChart` and `barChart` components rendered from agent tool calls
- **A2UI**: both fixed-schema flight cards and fully dynamic dashboards generated by a secondary LLM
- **Human-in-the-Loop**: a meeting-time picker that pauses the agent until the user responds
- **Frontend tools**: theme toggle plus an app-mode toggle that reveals a todo canvas with shared state

## How to Interact

Click any suggestion chip, or type your own prompt. For example:

- "Show me a pie chart of our revenue distribution by category"
- "Find flights from SFO to JFK for next Tuesday"
- "Show me a sales dashboard with total revenue, new customers, and conversion rate"
- "Enable app mode and add three todos about learning CopilotKit"

## Technical Details

- `CopilotKit` is wired to a dedicated runtime (`/api/copilotkit-beautiful-chat`) with `a2ui={{ catalog: demonstrationCatalog }}` and `openGenerativeUI` enabled, selecting the `beautiful-chat` agent
- `useComponent` registers `pieChart` / `barChart`; `useHumanInTheLoop` registers `scheduleTime`; `useFrontendTool` registers `toggleTheme`, `enableAppMode`, and `enableChatMode`; `useDefaultRenderTool` draws reasoning cards for any other backend tool
- `useConfigureSuggestions` seeds the starter chips; `CopilotChat` renders the chat surface alongside an `ExampleCanvas` that swaps in when app mode is enabled
- The agent exposes `query_data`, `manage_todos`/`get_todos`, `search_flights` (A2UI fixed schema), and `generate_a2ui` (dynamic schema via a secondary `gpt-4.1` call), with `StateStreamingMiddleware` streaming `todos` into shared state as `manage_todos` runs
