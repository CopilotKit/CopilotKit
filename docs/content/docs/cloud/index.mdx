## Introduction

CopilotKit Cloud provides the backend infrastructure for building Agent native applications. It offers:

A core runtime that handles LLM communication and agent orchestration, eliminating the need for custom deployment
Built-in support for agent frameworks like LangGraph, making it seamless to connect your frontend with sophisticated AI agents

Enterprise features include secure authentication and content moderation guardrails, with more capabilities coming soon.

Cloud-specific features are denoted by the `_c` suffix, indicating functionality exclusive to CopilotKit's cloud platform

## Features

- **Authenticated Actions**: Propagate authentication state to your backend tools and services
- **Guardrails**: Content moderation and topic control for AI interactions
