## Prerequisites

Before you begin, you'll need the following:

- **.NET 9.0 SDK**
- **Azure OpenAI endpoint and deployment** (or compatible chat client)
- **Node.js 20+** for the frontend and CopilotKit CLI

            <Step>
                ### Create and run a .NET AG-UI server

                #### Create the project and add packages
                ```bash
                dotnet new web -n AGUIServer
                cd AGUIServer
                dotnet add package Microsoft.Agents.AI.Hosting.AGUI.AspNetCore
                dotnet add package Microsoft.Agents.AI.OpenAI
                dotnet add package Azure.AI.OpenAI
                dotnet add package Azure.Identity
                ```

                #### Minimal server setup
                ```csharp title="Program.cs"
                using Azure.AI.OpenAI;
                using Azure.Identity;
                using Microsoft.Agents.AI;
                using Microsoft.Agents.AI.Hosting.AGUI.AspNetCore;

                var builder = WebApplication.CreateBuilder(args);
                builder.Services.AddHttpClient().AddLogging();
                var app = builder.Build();

                string endpoint = builder.Configuration["AZURE_OPENAI_ENDPOINT"]!;
                string deployment = builder.Configuration["AZURE_OPENAI_DEPLOYMENT_NAME"]!;

                app.MapAGUIAgent("/", (messages, tools, context, forwardedProps) =>
                {
                    return new AzureOpenAIClient(new Uri(endpoint), new DefaultAzureCredential())
                        .GetChatClient(deployment)
                        .CreateAIAgent(name: "AGUIAssistant");
                });

                await app.RunAsync();
                ```

                #### Configure and run
                ```bash
                # macOS/Linux
                export AZURE_OPENAI_ENDPOINT="https://your-resource.openai.azure.com/"
                export AZURE_OPENAI_DEPLOYMENT_NAME="gpt-4o-mini"

                # Windows (PowerShell)
                $env:AZURE_OPENAI_ENDPOINT="https://your-resource.openai.azure.com/"
                $env:AZURE_OPENAI_DEPLOYMENT_NAME="gpt-4o-mini"

                dotnet run --urls "http://localhost:5100"
                ```

                    <Step>
                        ### Add a remote endpoint for your AG-UI server
                        Using Copilot Cloud, create a remote endpoint that connects to your locally running or deployed AG-UI server.

                        #### Tunnel your local server
                        First, log in to [Copilot Cloud](https://cloud.copilotkit.ai), then authenticate the CLI:
                        ```bash
                        npx copilotkit@latest login
                        ```
                        Start a tunnel to your local AG-UI server (port 5100):
                        ```bash
                        npx copilotkit@latest dev --port 5100
                        ```

                        #### Configure the remote endpoint in Copilot Cloud
                        - Endpoint URL: your AG-UI server URL (or the tunnel URL)
                        - Optional: Authorization header if your server requires it

                        <Accordions className="mb-4">
                            <Accordion title="Having trouble?">
                                - Ensure your AG-UI server is reachable on the specified port (default 5100)
                                - On Windows, verify these entries exist in your hosts file:
                                ```bash
                                127.0.0.1 localhost
                                ::1       localhost
                                ```
                            </Accordion>
                        </Accordions>
                    </Step>
                    <Step>
                        ### Setup your CopilotKit provider
                        Wrap your application with the [`<CopilotKit>`](/reference/components/CopilotKit) provider and configure it to use Copilot Cloud.

                        ```tsx title="app/layout.tsx"
                        import { CopilotKit } from "@copilotkit/react-core";

                        export default function RootLayout({ children }: { children: React.ReactNode }) {
                          return (
                            <CopilotKit
                              publicApiKey={process.env.NEXT_PUBLIC_COPILOT_CLOUD_PUBLIC_API_KEY}
                              properties={{ authorization: userToken }}
                            >
                              {children}
                            </CopilotKit>
                          );
                        }
                        ```
