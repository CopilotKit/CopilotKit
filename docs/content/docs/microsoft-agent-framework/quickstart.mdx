## Prerequisites

Before you begin, you'll need the following:

- [**.NET 9.0 SDK**](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)
- [**Node.js 20+**](https://nodejs.org/en/download)
- [**GitHub Personal Access Token**](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens#creating-a-personal-access-token-classic) (for GitHub Models API)
- Optional: [**Azure OpenAI endpoint and deployment**](https://learn.microsoft.com/en-us/azure/ai-foundry/how-to/deploy-models-openai) (if using Azure OpenAI instead of GitHub Models)

                        ```bash
                        dotnet new web -n AGUIServer
                        cd AGUIServer
                        dotnet add package Microsoft.Agents.AI.AGUI
                        dotnet add package Microsoft.Agents.AI.Hosting.AGUI.AspNetCore
                        dotnet add package Microsoft.Extensions.AI.OpenAI
                        dotnet add package OpenAI
                        ```

                        ```csharp title="Program.cs (excerpt)"
                        using Microsoft.Agents.AI;
                        using Microsoft.Agents.AI.AGUI;
                        using Microsoft.Agents.AI.Hosting.AGUI.AspNetCore;
                        using Microsoft.Extensions.AI;
                        using OpenAI;

                        var builder = WebApplication.CreateBuilder(args);
                        var app = builder.Build();

                        app.MapAGUIAgent("/", (messages, tools, context, forwardedProps) => {
                          var githubToken = builder.Configuration["GitHubToken"]!; // dotnet user-secrets set GitHubToken "<token>"
                          var openAI = new OpenAIClient(new System.ClientModel.ApiKeyCredential(githubToken), new OpenAIClientOptions { Endpoint = new Uri("https://models.inference.ai.azure.com")});
                          var chat = openAI.GetChatClient("gpt-4o-mini").AsIChatClient();
                          var agent = new ChatClientAgent(chat, name: "SampleAgent", description: "A helpful assistant.");
                          return new ChatClientAGUIAgent(agent);
                        });

                        app.Run("http://localhost:8000");
                        ```

                        ```bash
                        # Set GitHub token as user secret and run

                        dotnet run
                        ```
