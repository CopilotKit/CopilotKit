<DeprecationNotice
  deprecatedName="useRenderToolCall"
  replacementName="useToolRenderingResolver"
  replacementHref="/reference/v2/hooks/useToolRenderingResolver"
  guideHref="/migration-guides/migrate-use-tool-rendering-resolver"
>
  `useRenderToolCall` will continue to work during the deprecation window, but
  it logs a development warning. New custom or headless chat UIs should call
  `useToolRenderingResolver` instead.
</DeprecationNotice>

## Replacement

```tsx
import { useToolRenderingResolver } from "@copilotkit/react-core/v2";

const resolveToolRendering = useToolRenderingResolver();
```

`useToolRenderingResolver` is the same low-level resolver with a clearer name: it resolves an existing renderer for a tool call. It does not register a renderer and it does not execute a tool handler.

## Before / After

**Before:**

```tsx
import { useRenderToolCall } from "@copilotkit/react-core/v2";

function ToolCalls({ toolCall, toolMessage }) {
  const renderToolCall = useRenderToolCall();

  const renderedToolElement = renderToolCall({
    toolCall,
    toolMessage,
  });

  return renderedToolElement ? (
    <div className="tool-call">{renderedToolElement}</div>
  ) : null;
}
```

**After:**

```tsx
import { useToolRenderingResolver } from "@copilotkit/react-core/v2";

function ToolCalls({ toolCall, toolMessage }) {
  const resolveToolRendering = useToolRenderingResolver();

  const renderedToolElement = resolveToolRendering({
    toolCall,
    toolMessage,
  });

  return renderedToolElement ? (
    <div className="tool-call">{renderedToolElement}</div>
  ) : null;
}
```

## Codemod

Run the migration codemod from your app:

```bash
npx jscodeshift -t https://raw.githubusercontent.com/CopilotKit/CopilotKit/main/codemods/migrate-use-tool-rendering-resolver.ts --parser=tsx --extensions=tsx,ts ./src
```

The codemod updates imports, unaliased call sites, and v2 re-exports from `@copilotkit/react-core/v2`. It intentionally does not touch the v1 `useRenderToolCall` hook exported from `@copilotkit/react-core`.
