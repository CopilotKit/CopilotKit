## Overview

The `useAgent` hook provides direct access to your Mastra agent from any React component. It gives you real-time access to the agent's state, messages, execution status, and allows you to subscribe to custom events.

This enables you to build custom agent dashboards, monitoring tools, and interactive features that respond to your agent's behavior.

### Workflow-Specific State

If your Mastra agent tracks workflow stages, you can show contextual UI:

```tsx title="page.tsx"
export function WorkflowStatus() {
  const { agent } = useAgent();

  // [!code highlight:1]
  const currentStep = agent.state.currentStep;

  return (
    <div>
      {/* [!code highlight:6] */}
      {currentStep === "data_collection" && (
        <div className="alert">Agent is collecting data...</div>
      )}
      {currentStep === "processing" && (
        <div className="alert">Agent is processing results...</div>
      )}
    </div>
  );
}
```
