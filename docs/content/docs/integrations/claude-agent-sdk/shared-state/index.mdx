## What is shared state?

The Claude Agent SDK adapter maintains a shared state that seamlessly connects your UI with the agent's execution via the AG-UI protocol. This shared state system allows you to:

- Display the agent's current progress and intermediate results
- Update the agent's state through UI interactions
- React to state changes in real-time across your application

The adapter automatically provides an `ag_ui_update_state` tool that the agent can use to update shared state, and injects the current state into the agent's context.

## When should I use this?

State streaming is perfect when you want to facilitate collaboration between your agent and the user. Any state that your Claude Agent SDK agent
updates via the `ag_ui_update_state` tool will be automatically shared with the UI. Similarly, any state that the user updates in the UI will be automatically reflected in the agent's context.

This allows for a consistent experience where both the agent and the user are on the same page.
