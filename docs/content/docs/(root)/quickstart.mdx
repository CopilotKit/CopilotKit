## Install Dependencies

<Callout type="warn">
  Next.js 15 with `--turbopack` option (i.e. `"dev": "next dev --turbopack"`) is
  currently not supported with CopilotKit.
</Callout>

First, install the latest pre-release CopilotKit dependencies.

```package-install
npm install @copilotkit/react-ui @copilotkit/react-core
```

In order to use CopilotKit, we'll need to configure the `CopilotKit` provider.
