## Recent Updates (as of October 30, 2025)

### [CopilotKit v1.50](/whats-new/v1-50)
CopilotKit v1.50 is coming soon, with brand new interfaces, streamlined internals, and no breaking changes.

### [A2A and MCP Handshake](/whats-new/a2a-mcp-handshake)
AG-UI now includes handshakes with both A2A and MCP protocols, enabling seamless interoperability across the full agentic stack.

### [A2UI - Google's Generative UI Spec](/whats-new/a2ui)
CopilotKit now supports A2UI, Google's declarative, LLM-friendly Generative UI specification.
