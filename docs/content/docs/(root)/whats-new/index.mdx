## A2UI Launch Day

CopilotKit has been working with Google on A2A and A2UI integrations, and is proud to be a launch partner for A2UI.

Check out the [A2UI composer](https://a2ui-editor.ag-ui.com/) (written by CopilotKit engineers), which you can use to create or find widgets to use in the CopilotKit [A2A integration](./a2a). [Read more →](/whats-new/a2ui-launch)

## CopilotKit v1.50

CopilotKit v1.50 is now available, with brand new interfaces, streamlined internals, and no breaking changes. [Read more →](/whats-new/v1-50)

## Generative UI Spec Support

CopilotKit now supports all major Generative UI specifications - A2UI, Open-JSON-UI, and MCP-UI. [Read more →](/whats-new/generative-ui-spec-support)

## A2A and MCP Handshake

AG-UI now includes handshakes with both A2A and MCP protocols, enabling seamless interoperability across the full agentic stack. [Read more →](/whats-new/a2a-mcp-handshake)
