# Enterprise Features

CopilotKit provides enterprise-grade features for production applications that require advanced monitoring, analytics, and compliance capabilities.

## Available Features

### Observability

Monitor and track user interactions with your CopilotKit applications.

- **Event Hooks**: Track chat events like message sending, feedback, and user interactions
- **Error Monitoring**: Monitor system errors and performance issues
- **Analytics Integration**: Connect with your existing analytics platforms

[Learn more about Observability →](./observability)

## Getting Started

All enterprise features require a CopilotKit Cloud account with a valid `publicApiKey`.

1. **Sign up for CopilotKit Cloud**: [https://cloud.copilotkit.ai](https://cloud.copilotkit.ai)
2. **Get your API key**: Copy your public API key from the dashboard
3. **Configure your app**: Add the `publicApiKey` to your `<CopilotKit>` provider

```tsx
<CopilotKit publicApiKey="YOUR_PUBLIC_API_KEY">{/* Your app */}</CopilotKit>
```
