## Prebuilt Components

CopilotKit provides a set of prebuilt components for you to use in your own custom UI components.

## Chat
This is the most basic component you can use to connect to an LLM or agent. It is the main
component to utilize when you want to embed CopilotKit in your own UI.

<ChatPreview>
  ```tsx
  import { CopilotKit } from "@copilotkit/react-core";
  import { CopilotChat } from "@copilotkit/react-ui";
  import "@copilotkit/react-ui/styles.css";

  const Chat = () => {
    return (
      <CopilotKit
        // https://cloud.copilotkit.ai
        publicApiKey={"your-public-cloud-key"}
        // or for self-hosted
        runtimeUrl={"" /*https://example.com/runtime*/}
      >
        <CopilotChat 
          className="h-96"
          labels={{
            initial: "Hey! I'm connected to an LLM through CopilotKit ðŸª",
          }}
        />
      </CopilotKit>
    )
  }
  ```
</ChatPreview>
